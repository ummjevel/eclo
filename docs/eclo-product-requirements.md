# Eclo - 제품 요구사항 정의서 (PRD)

> **제품명**: Eclo  
> **제품 유형**: Multilingual On-Device TTS & Voice Cloning Mac Application  
> **버전**: 1.0  
> **최종 수정일**: 2026-01-11

---

## 1. 제품 개요

### 1.1 제품 비전

Eclo는 Apple Silicon Mac에서 완전히 오프라인으로 동작하는 다국어 텍스트-투-스피치(TTS) 및 음성 복제(Voice Cloning) 애플리케이션입니다. 9개 언어를 지원하며, 3초의 음성 샘플만으로 사용자의 목소리를 복제할 수 있습니다.

### 1.2 핵심 가치 제안 (Value Proposition)

| 기존 솔루션의 문제 | Eclo의 해결책 |
|-------------------|--------------|
| 클라우드 의존 (프라이버시 우려) | ✅ 100% 오프라인 처리 |
| 월 구독료 ($5-22/월) | ✅ 일회성 구매 |
| 영어 중심 | ✅ 9개 언어 지원 (아시아 언어 포함) |
| Voice Cloning 미지원/고가 | ✅ Zero-shot Voice Cloning 포함 |

### 1.3 타겟 사용자

| 사용자 그룹 | 사용 시나리오 |
|------------|-------------|
| 콘텐츠 크리에이터 | 다국어 나레이션, YouTube 더빙 |
| 언어 학습자 | 발음 연습, 청취 자료 생성 |
| 접근성 필요 사용자 | 시각장애인, 난독증, ADHD |
| 개발자/연구자 | 프로토타입 음성 생성 |
| 프라이버시 중시 사용자 | 민감한 텍스트 음성 변환 |

## 3. 기능 요구사항

### 3.1 핵심 기능 (MVP - v1.0)

#### FR-001: 다국어 텍스트-투-스피치

| 항목 | 상세 |
|------|------|
| **설명** | 입력된 텍스트를 선택한 언어의 자연스러운 음성으로 변환 |
| **지원 언어** | 중국어, 영어, 일본어, 한국어, 독일어, 스페인어, 프랑스어, 이탈리아어, 러시아어 (9개) |
| **입력** | 텍스트 (최대 5,000자) |
| **출력** | WAV, MP3 오디오 파일 |
| **우선순위** | 🔴 필수 |

```
사용자 스토리:
"사용자로서, 한국어 텍스트를 입력하면 자연스러운 한국어 음성으로 
변환된 오디오 파일을 받고 싶다."
```

#### FR-002: Zero-shot Voice Cloning

| 항목 | 상세 |
|------|------|
| **설명** | 3-10초 음성 샘플로 해당 목소리를 복제하여 TTS 적용 |
| **입력** | 참조 음성 파일 (WAV/MP3, 3-10초) + 텍스트 |
| **출력** | 복제된 목소리로 생성된 오디오 |
| **모드** | Cross-lingual (기본), Zero-shot (텍스트 전사 제공 시) |
| **우선순위** | 🔴 필수 |

```
사용자 스토리:
"사용자로서, 내 목소리 샘플 5초를 업로드하면 그 목소리로 
다른 언어(예: 영어)를 말하는 오디오를 생성하고 싶다."
```

#### FR-003: 음성 스타일/감정 제어 (Instruct Mode)

| 항목 | 상세 |
|------|------|
| **설명** | 자연어 지시로 음성 스타일 조절 |
| **지원 지시** | 속도 (빠르게/천천히), 감정 (행복/슬픔/차분), 톤 (전문적/친근한) |
| **입력** | 텍스트 + 스타일 지시문 |
| **우선순위** | 🟡 중요 |

```
사용자 스토리:
"사용자로서, '천천히 그리고 차분하게 말해줘'라고 지시하면 
그에 맞는 스타일로 음성이 생성되길 원한다."
```

#### FR-004: 음성 변환 (Voice Conversion)

| 항목 | 상세 |
|------|------|
| **설명** | 기존 오디오의 목소리를 다른 목소리로 변환 |
| **입력** | 원본 오디오 + 타겟 목소리 샘플 |
| **출력** | 타겟 목소리로 변환된 오디오 |
| **우선순위** | 🟢 선택 (v1.1) |

struct UserConsent {
    let id: UUID
