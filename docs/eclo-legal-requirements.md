# Eclo - 법적 보호 및 사용자 동의 요구사항

> **문서 목적**: Voice Cloning 기능이 포함된 Multilingual On-Device TTS Mac 앱 "Eclo" 개발 시 개발자 법적 책임을 최소화하기 위한 요구사항 정의
> 
> **최종 수정일**: 2026-01-11  
> **버전**: 1.0

---

## 1. 법적 배경 요약

### 1.1 관련 법률 현황 (2024-2025)

| 법률/규정 | 관할권 | 핵심 내용 | 개발자 면책 여부 |
|----------|--------|----------|----------------|
| **ELVIS Act** | 테네시 (2024) | AI 음성 복제 무단 사용 형사처벌, 민사 구제 | ⚠️ 불명확 |
| **Act 35** | 펜실베이니아 (2025) | 딥페이크 생성/배포 형사처벌 | ✅ "의도적 촉진" 아니면 면책 |
| **HB 1205** | 워싱턴 (2025) | 위조 디지털 초상 사용 형사처벌 | ✅ 삭제 요청 신속 대응 시 면책 |
| **Section 230** | 연방법 | 온라인 중개자 제3자 콘텐츠 면책 | ⚠️ AI 생성 콘텐츠는 불확실 |
| **GDPR** | EU | 음성 = 생체정보, 명시적 동의 필요 | ❌ 엄격한 규제 |

### 1.2 핵심 판례: Lehrman v. Lovo, Inc. (2024)

- **사건**: 성우들이 AI 음성 서비스 회사 Lovo를 무단 음성 사용으로 제소
- **결과**: 연방법(저작권법, Lanham Act) 청구는 기각, 뉴욕주 퍼블리시티권 청구는 진행 허용
- **시사점**: 주법에 따라 책임 범위가 달라질 수 있음

## 6. 신고 및 삭제 요청 대응 시스템

### 6.1 신고 채널

```
이메일: abuse@eclo.app (예시)
응답 시간: 24-72시간 이내
```

### 6.2 대응 프로세스

```
1. 신고 접수
   └─> 자동 확인 이메일 발송

2. 검토 (24시간 이내)
   └─> 신고 내용 확인
   └─> 해당 사용자 활동 검토

3. 조치 (48시간 이내)
   ├─> 경고: 경미한 위반
   ├─> 일시 정지: 반복 위반
   └─> 영구 차단: 심각한 위반

4. 결과 통보
   └─> 신고자 및 피신고자에게 결과 통보
```

### 6.3 DMCA 스타일 삭제 요청 양식

```
필수 정보:
- 신고자 연락처
- 침해된 권리 설명 (예: 음성 무단 사용)
- 침해 콘텐츠 설명
- 진술서: "본인은 해당 콘텐츠가 본인의 권리를 침해한다고 
  선의로 믿습니다"
- 서명 (전자 서명 가능)
```

## 8. 지역별 추가 고려사항

### 8.1 EU (GDPR)

```
추가 요구사항:
- 음성 데이터 = 생체정보 → 명시적 동의 필수
- 데이터 처리 동의 별도 획득
- 삭제 요청 권리 (잊힐 권리) 보장
- 데이터 처리 목적 명확히 고지
```

### 8.2 미국 (주별 상이)

```
일리노이 (BIPA):
- 생체정보 수집 전 서면 동의 필수
- 위반 시 $1,000-$5,000/건 손해배상

캘리포니아 (CCPA):
- 개인정보 수집 고지 의무
- 판매 거부 권리 보장
```

### 8.3 한국

```
개인정보보호법:
- 음성 = 개인정보 → 동의 필요
- 목적 외 사용 금지
- 파기 의무

초상권/성명권:
- 명확한 법률 부재
- 민법상 불법행위로 처리 가능
```
