# Voice Samples

이 디렉토리에 프리셋 음성 샘플 파일을 넣어주세요.

## 필요한 파일

CosyVoice3 모델은 레퍼런스 오디오가 필요합니다. 각 언어별로 3-10초 길이의 WAV 파일을 준비해주세요.

| 파일명 | 설명 |
|--------|------|
| `korean_female.wav` | 한국어 여성 음성 (Sora) |
| `korean_male.wav` | 한국어 남성 음성 (Minho) |
| `english_female.wav` | 영어 여성 음성 (Emma) |
| `english_male.wav` | 영어 남성 음성 (James) |
| `japanese_female.wav` | 일본어 여성 음성 (Yuki) |
| `japanese_male.wav` | 일본어 남성 음성 (Kenji) |
| `chinese_female.wav` | 중국어 여성 음성 (Mei) |
| `chinese_male.wav` | 중국어 남성 음성 (Wei) |

## 오디오 요구사항

- 포맷: WAV (권장) 또는 MP3
- 길이: 3-10초
- 샘플레이트: 16kHz 이상
- 비트: 16bit 이상
- 채널: 모노 권장

## 음성 샘플 수집 방법

1. **직접 녹음**: 마이크로 직접 녹음
2. **공개 데이터셋**: LibriSpeech, Common Voice 등에서 수집
3. **TTS 생성**: 다른 TTS 서비스에서 생성 (라이선스 확인 필요)

## 주의사항

- 저작권이 있는 음성은 사용하지 마세요
- 타인의 음성은 동의 없이 사용하지 마세요
- 상업적 배포 시 라이선스를 확인하세요
